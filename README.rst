thealphabetalab_branding plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/thealphabetalab/tutor-contrib-thealphabetalab-branding

Usage
*****

.. code-block:: bash

    tutor plugins enable thealphabetalab_branding
